Soloscaler's five core values are designed to function as a decision-making framework. Each value should be able to resolve a product, hiring, or content decision when invoked. If a value cannot settle a real argument, it is decoration.

**1. Leverage Over Labor** — "Find the lever, not the grind. If the answer is 'hire more people,' we haven't looked hard enough." This value drives product architecture (automate before staffing), consulting recommendations (systems over headcount), and content tone (solutions, not hustle).

**2. Radical Honesty** — "'You don't need this' is a valid recommendation. Trust is the product; everything else is a feature." This is the most commercially risky value. Telling a potential client they do not need you reduces short-term revenue. But it is the strongest trust-building move in a market full of agencies optimizing for retainer length.

**3. Builder's Mentality** — "Strategy without execution is a hobby. Every engagement produces something tangible." This constrains the consulting practice: no pure advisory engagements. If it does not result in a working system, it is not a Soloscaler engagement.

**4. Right-Sized by Design** — "Simplicity isn't a limitation, it's a design philosophy. Reject the enterprise-lite paradigm." This constrains product development: never build by removing features from a bigger product. Always start from the small business workflow.

**5. Self-Sufficiency as the Goal** — "The best engagement ends with the client not needing us anymore." This is the boldest value because it caps customer lifetime value by design. It also creates the strongest referral engine — clients who become self-sufficient become evangelists.

In the product-brand fit analysis, each of the six portfolio products maps to one of these values, proving the values are operational, not aspirational.
